Bootstrapper¶

class bootstrapper.Bootstrapper(n_jobs=-1, bootstrap_count=10000)[source]¶

The entry point of all Bootstrapper features.

Parameters:
  • n_jobs (int, optional) – How many process workers are used. -1 means the CPU counts available. Defaults to -1.
  • bootstrap_count (int, optional) – How many bootstrap samples are drawn. Defaults to 10_000.
Raises:
  • ValueError – When n_jobs is not positive integer or -1.
  • ValueError – When bootstrap_count is not positive integer.
